Git Respositories > 프로젝트명 > Branches > Local > master 에서 마우스 오른쪽 'reset' 클릭 후 hard 선택

프로젝트명에서 마우스 오른쪽 누르고 Pull 선택



명령어는

cd 프로젝트 디렉토리

> 
git reset --hard<br>
git pull
